# myFirstCRUD
Esse é o meu primeiro exemplo de CRUD, utilizando MongoDB, seguindo o curso do Luis Otavio Miranda.

*******************************************************

PS2: Não se esqueça de rodar o comando "npm i" para instalar o node-modules.
